Elements
<ELEMENT element-name EMPTY> - empty element
<!ELEMENT element-name (#PCDATA)> - elements with Parsed Character Data
<!ELEMENT element-name ANY> - elements with any contents
<!ELEMENT element-name (child1,child2,...)> - elements with children
<!ELEMENT element-name (child-name)> - only one occurrence of an element
<!ELEMENT element-name (child-name+)> - minimum one occurrence of an element
<!ELEMENT element-name (child-name*)> - zero or zore occurrences of an element
<!ELEMENT element-name (child-name?)> - zero or one occurrences of an element
<!ELEMENT note (to,from,header,(message|body))> - "message" or "body"
Attributes
<!ATTLIST person number CDATA> - attribute value is character data
<!ATTLIST person number CDATA #REQUIRED> - attribute required
<!ATTLIST contact fax CDATA #IMPLIED> - attribute not required
<!ATTLIST payment type (check|cash)> - attribute value may be "check" or "cash"
